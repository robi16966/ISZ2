# ISZ2

1. Vytvorte hierarchiu tried domacich zvierat (pes, macka, kon... zakladna trieda -DomaceZviera) a
divokych zvierat (Puma, Lev, …). kazde zviera vydava specificky zvuk (Pes : Haf) - vyjadrite tuto
skutocnost v hierarchii tried pomocou interface. Hlasenia divokych zvierat sa budu vo
vseobecnosti lisit (budu uppercase alebo budu mat vseobecni prefix alebo sufix).
2. Vytvorte prototyp aplikaciu ktora nacita zoznam domacich zvierat zo suboru a uloži ich do
databazi. Nasledne uzivatel moze v prikazovom riadku zadat prislusne zviera a program mu
zobrazi zvuk ktory vydava alebo chybovu hlasku ak sa zviera nenachadza v systeme. Format
suboru si mozete zvolit lubovolne (xml,json,plaintext).<br>
Príklad riadku suboru:
1. Pes[DomaceZviera] : Haf
2. Lev[DivokeZviera] : Roooar
